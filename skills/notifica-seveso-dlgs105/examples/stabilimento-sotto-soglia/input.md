# Input - officina con piccole quantita' di sostanze pericolose

Un'officina meccanica detiene modeste quantita' di sostanze pericolose (oli,
solventi, bombole di gas tecnici). Dal censimento risulta che, per ciascuna
sostanza e categoria dell'allegato 1, le quantita' presenti sono **inferiori
alla colonna 2** (soglia inferiore) e, applicando la regola della sommatoria,
non si raggiunge il superamento di alcuna categoria.

Domanda: l'officina e' uno stabilimento Seveso? Deve fare la notifica del
D.Lgs. 105/2015?
