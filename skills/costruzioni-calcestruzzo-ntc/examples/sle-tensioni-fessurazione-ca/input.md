# Input - Verifiche SLE di tensione e fessurazione di una trave in c.a.

Un progettista deve inquadrare le **verifiche agli stati limite di esercizio** di una **trave in cemento
armato** di un edificio in **condizioni ambientali ordinarie**, con armatura ordinaria (poco sensibile alla
corrosione). Il calcestruzzo è di classe **C25/30** (fck = 25 MPa) e l'acciaio ha fyk = 450 MPa.

Chiede: quali sono i **limiti di tensione** del calcestruzzo e dell'acciaio in esercizio secondo le NTC 2018
§4.1? Quale **stato limite di fessurazione** e quale **apertura limite** (w1, w2 o w3) vanno considerati?
