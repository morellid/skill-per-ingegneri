# Input - manuale per una macchina destinata a operatori non professionali

Un fabbricante immette sul mercato una piccola macchina da giardino destinata a
**consumatori (operatori non professionali)**. Chiede: nella redazione delle
istruzioni per l'uso, di cosa deve tenere conto in piu' rispetto a una macchina
industriale, e quali voci del contenuto minimo sono particolarmente rilevanti?
Cita l'allegato III del Reg. (UE) 2023/1230.
