# Input - in che lingua possono essere le istruzioni di manutenzione?

Un fabbricante fornisce una linea industriale a uno stabilimento italiano. Le
istruzioni di **manutenzione** sono destinate a essere usate solo da **personale
specializzato incaricato dal fabbricante**. Chiede: queste istruzioni di
manutenzione devono essere tradotte in italiano come il resto del manuale, o e'
ammessa una sola lingua? E il manuale puo' essere fornito solo in PDF? Cita il
Reg. (UE) 2023/1230.
