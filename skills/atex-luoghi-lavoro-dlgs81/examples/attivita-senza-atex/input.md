# Input - deposito di esplosivi

Un'impresa gestisce un deposito in cui si effettuano **stoccaggio e
manipolazione di esplosivi**. Il responsabile chiede se deve redigere il
documento sulla protezione contro le esplosioni (DPCE) ai sensi del Titolo XI
del D.Lgs. 81/2008.

Domanda: si applica il Titolo XI (ATEX luoghi di lavoro) a questa attivita'?
