# Input - direttore dell'esecuzione in un appalto di servizi

Sono il RUP di una stazione appaltante per un appalto di **servizi** di pulizia pluriennale.
Devo nominare un **direttore dell'esecuzione del contratto** separato oppure le funzioni le
svolgo io come RUP? In quali casi il **DEC deve essere diverso dal RUP**? Posso farmi aiutare
da assistenti? Cita l'art. 114 del D.Lgs. 36/2023.
