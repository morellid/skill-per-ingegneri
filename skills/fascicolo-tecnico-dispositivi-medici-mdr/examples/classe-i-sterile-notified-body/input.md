# Input - dispositivo di classe I sterile: serve un organismo notificato?

Un fabbricante immette sul mercato una **medicazione avanzata** classificata come
**dispositivo di classe I** ma fornita **sterile**. Chiede: puo' procedere in
autodichiarazione come per la classe I "base", oppure serve un organismo
notificato? E per quali aspetti? Cita l'art. 52 MDR.
