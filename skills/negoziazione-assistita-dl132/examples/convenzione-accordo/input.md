# Input - requisiti della convenzione ed effetti dell'accordo

Due parti, ciascuna con il proprio avvocato, vogliono avviare una negoziazione
assistita e chiedono:

1. Quali requisiti deve avere la convenzione (termine, forma, oggetto)?
2. Una volta raggiunto, che valore ha l'accordo? Può essere usato per un pignoramento?
3. L'avvocato che ha partecipato può poi impugnare l'accordo?
