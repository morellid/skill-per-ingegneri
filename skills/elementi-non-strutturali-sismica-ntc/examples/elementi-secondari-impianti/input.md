# Input - Elementi secondari e impianto pesante

Un progettista, secondo le NTC 2018 §7.2.3-7.2.4, deve:
- decidere se alcuni pilastri possono essere considerati **elementi strutturali secondari** (trascurandone
  rigidezza/resistenza orizzontale nell'analisi);
- valutare un **impianto tecnologico pesante** (una macchina) appeso a un solaio, il cui peso è pari a circa il
  **35% del carico permanente del campo di solaio**.

Chiede: quali limiti valgono per considerare gli elementi come secondari? L'impianto rientra nelle prescrizioni
ordinarie del §7.2.4?
