# Nota

Esempio ancorato al **§7.3.5** (analisi dinamica con integrazione al passo) delle NTC 2018, trascritto in
`references/fonti/ntc2018-par-7-3-5.md`.

Punti verificati sul testo:

- Si applicano **simultaneamente le due componenti orizzontali** (piu' la verticale, ove necessario).
- Con **almeno 3 storie temporali** si usano i **valori piu' sfavorevoli**; con **almeno 7 diverse storie
  temporali** si usa la **media dei valori piu' sfavorevoli** (§7.3.5).
- Per la **variabilita' spaziale del moto**: storie temporali differenziate coerenti per ciascun vincolo, oppure
  moto sincrono con effetti pseudo-statici (§3.2.4).

La regola 3/7 e' lettura diretta del §7.3.5 (illustrazione). La skill **non** genera le storie ne' esegue
l'analisi.
