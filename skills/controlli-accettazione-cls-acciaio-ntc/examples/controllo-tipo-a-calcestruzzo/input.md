# Input - Controllo di accettazione del calcestruzzo (tipo A)

Un Direttore dei Lavori deve impostare il **controllo di accettazione** del calcestruzzo di una struttura di
fondazione. È previsto il getto di **250 m³** di una **miscela omogenea** in classe **C25/30** (resistenza
caratteristica di progetto **Rck = 30 N/mm²**), in più giornate.

Chiede: quale **tipo di controllo** si applica e con quanti **prelievi**? Quali **disuguaglianze** devono
essere verificate per accettare il calcestruzzo secondo le NTC 2018 §11.2.5? Se i prelievi dessero **Rcm28 =
36,0 N/mm²** e **Rc,min = 32,5 N/mm²**, il controllo è positivo?
