# Input - Categoria di rischio e procedura di conformità di un imbracatura anticaduta

Un fabbricante produce un'**imbracatura anticaduta** destinata a proteggere dal rischio di **caduta
dall'alto**. Vuole immettere il prodotto sul mercato con la marcatura CE secondo il Reg. UE 2016/425.

Chiede: in quale **categoria di rischio** rientra il DPI? Quale **procedura di valutazione della
conformità** deve seguire e interviene un **organismo notificato**?
