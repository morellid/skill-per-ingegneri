# Note - PMI con ISO 45001

- Fonte: `references/fonti/mog-231-sicurezza-lavoro.md` (art. 30 c.2-c.5-ter D.Lgs
  81/2008; artt. 6-7 D.Lgs 231/2001).
- Punto chiave: la presunzione di conformita' (art. 30 c.5) vale **"per le parti
  corrispondenti"**, non come equivalenza automatica: mancano tipicamente sistema
  disciplinare, registrazioni, articolazione di funzioni e il raccordo 231 (OdV,
  mappatura reato).
- Utile citare le semplificazioni per PMI (c.5-bis/5-ter).
- L'esempio evidenzia un errore comune ("ho la ISO 45001 quindi ho il MOG 231").
