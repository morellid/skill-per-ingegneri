# Input - parziale difformità: si può evitare la demolizione?

In un intervento con permesso di costruire ci sono **scostamenti parziali** rispetto al progetto
(parziale difformità), ma la parte difforme è **collegata a parti conformi** e demolirla le
danneggerebbe. Si deve demolire per forza o si può applicare una **sanzione**? Cita l'art. 34 del
D.P.R. 380/2001.
