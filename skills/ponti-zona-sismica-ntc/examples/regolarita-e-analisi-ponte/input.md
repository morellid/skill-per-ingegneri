# Esempio — Input: regolarità KR e ammissibilità dell'analisi statica lineare

**Ponte rettilineo a travata continua** su 3 pile in c.a., in zona sismica, comportamento dissipativo, con
**q0 = 3,5**. Si valuta la direzione **longitudinale**.

Dati per la regolarità (momenti alla base e resistenti delle pile del sistema resistente al sisma):

| Pila | MEd,i [kNm] | MRd,i [kNm] |
|---|---|---|
| P1 | 4000 | 5000 |
| P2 | 4500 | 5000 |
| P3 | 3000 | 6000 |

Masse: massa efficace complessiva delle pile del sistema resistente = **1/8** della massa dell'impalcato.
Geometria: ponte rettilineo, obliquità trascurabile, B/L = 0,4.

Domande:
1. Il ponte è «regolare»? Quale KR devo usare?
2. È ammissibile l'analisi statica lineare in direzione longitudinale?
