# Note - formati della documentazione

- Fonte: `references/fonti/riuso-software-pa.md`, Linee guida AgID Allegato A, A.8.
- Punto chiave: la documentazione **tecnica** deve essere **versionabile riga per riga**
  (HTML/Markdown/reST/LaTeX); **PDF/DOCX/ODT non ammessi**. Il **manuale utente** ammette
  formati aperti/modificabili ma **non** il PDF.
- Errore comune: allegare PDF perché "già pronti".
