# Output atteso

Inquadramento documentale secondo le **NTC 2018 (D.M. 17 gennaio 2018), par. 2.4**. Nessun calcolo di spettro o
periodi di ritorno.

## 1. Vita nominale di progetto V_N (§2.4.1, Tab. 2.4.I)

Costruzione con **livelli di prestazioni ordinari** (tipo 2 della Tab. 2.4.I):

> **V_N = 50 anni** (valore minimo)

## 2. Classe d'uso (§2.4.2)

Una scuola prevede **affollamenti significativi**: ricade in

> **Classe III**

(Il §2.4.2 definisce la Classe III come costruzioni il cui uso preveda affollamenti significativi; è distinta
dalla Classe II dei normali affollamenti e dalla Classe IV delle funzioni pubbliche o strategiche importanti. La
classificazione puntuale, in presenza di funzioni strategiche/pubbliche particolari, resta valutazione del
progettista anche alla luce delle disposizioni regionali/nazionali applicabili.)

## 3. Coefficiente d'uso C_U e periodo di riferimento V_R (§2.4.3, [2.4.1], Tab. 2.4.II)

Per la **Classe III** il coefficiente d'uso è **C_U = 1,5** (Tab. 2.4.II). Il periodo di riferimento è:

> **V_R = V_N · C_U** = 50 · 1,5 = **75 anni**   [2.4.1]

## 4. Uso a valle (rinvio §3.2)

Il valore V_R = 75 anni è il periodo di riferimento da cui, tramite le probabilità di superamento di ciascuno
stato limite, si ricavano i **periodi di ritorno T_R** e lo **spettro** dell'azione sismica: passaggio del **§3.2**
(skill `spettro-risposta-ntc`), fuori scope di questa skill.

- NTC 2018 §2.4.3, formula [2.4.1] e Tab. 2.4.II - V_R = V_N · C_U, C_U = 1,5 per la Classe III → V_R = 75 anni.
