# Input - dispositivo Bluetooth: RED e scelta del modulo

Un fabbricante sviluppa un **sensore con modulo Bluetooth** (apparecchiatura
radio) alimentato a batteria. Chiede: si applica la RED? E se **non** esistono o
**non** vengono applicate le norme armonizzate per i requisiti di sicurezza/EMC
(art. 3.1), quale procedura di valutazione della conformita' deve usare e serve un
organismo notificato? Cita la RED.
