# Nota

Esempio ancorato al **§4.5.2.2** (classificazione degli elementi per foratura, Tab. 4.5.Ia), al
**§4.5.6.1** (resistenze di progetto fd = fk/γM, fvd = fvk/γM) e alla **Tab. 4.5.II** delle NTC 2018,
trascritti in `references/fonti/ntc2018-par-4-5.md`.

Punti verificati sul testo:

- Foratura **15% < Π ≤ 45%** → elemento **semipieno** (Tab. 4.5.Ia laterizio).
- **Tab. 4.5.II**: cat. I + malta a prestazione garantita → **2,0** (classe 1) / **2,5** (classe 2); cat. I
  + malta a composizione prescritta → 2,2 / 2,7; cat. II → 2,5 / 3,0.
- La distinzione **classe 1 / classe 2** dipende dai controlli in cantiere (supervisione, controllo
  ispettivo, controllo di malta/cls e dosaggio), come da nota alla Tab. 4.5.II.

La skill **non** calcola le resistenze in valore: fornisce solo l'inquadramento del coefficiente e del
criterio. I valori di fk/fvk provengono dal cap. 11.10 (non trattato) e dal progetto.
