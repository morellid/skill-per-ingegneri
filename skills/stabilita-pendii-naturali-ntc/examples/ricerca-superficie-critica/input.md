# Input - Scelta delle superfici di scorrimento da verificare

Un progettista deve impostare le **verifiche di stabilità** di due versanti diversi secondo le NTC 2018 §6.3.4:
- **Caso A**: un pendio **in frana**, di cui le indagini hanno accertato la superficie di scorrimento.
- **Caso B**: un pendio **non in frana**, senza superfici di scorrimento accertate.

Chiede: lungo quali **superfici di scorrimento** vanno eseguite le verifiche nei due casi? Come si individua la
superficie da porre a base della verifica?
