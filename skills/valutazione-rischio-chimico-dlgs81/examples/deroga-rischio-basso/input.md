# Input - Ufficio con uso di detergenti per pulizie

In un **ufficio** l'unico agente chimico presente è un **detergente multiuso** di uso domestico/civile,
utilizzato dall'addetto alle pulizie in **piccole quantità**, saltuariamente e con guanti. Il datore di
lavoro chiede: devo applicare tutte le misure specifiche e la sorveglianza sanitaria dell'art. 225, o
posso concludere per un rischio basso e irrilevante?
