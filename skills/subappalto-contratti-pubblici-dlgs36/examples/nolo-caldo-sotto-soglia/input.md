# Input - nolo a caldo in un appalto di lavori

Sono l'appaltatore di un appalto pubblico di **lavori** dell'importo di **1.000.000 euro**.
Voglio affidare a una ditta terza un **nolo a caldo** (escavatore con operatore) per un
importo di **15.000 euro**; l'incidenza del costo della manodopera su questa prestazione e'
di circa il **30%**. Devo chiedere l'autorizzazione al subappalto o basta una comunicazione?
Cita l'art. 119 del D.Lgs. 36/2023.
