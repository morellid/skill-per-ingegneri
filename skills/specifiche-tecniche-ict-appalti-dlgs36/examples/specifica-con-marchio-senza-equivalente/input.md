# Input - specifica tecnica ICT che nomina un prodotto senza "o equivalente"

Un ufficio di una PA sta redigendo il capitolato tecnico per la fornitura di un
**sistema di gestione documentale**. Nella bozza delle specifiche tecniche compare
la clausola:

> "Il sistema deve utilizzare il database **Oracle Database Enterprise Edition**
> versione 19c."

Nessun'altra indicazione e' presente. Il requisito e' tecnicamente giustificato da
una descrizione precisa dell'oggetto? La clausola e' conforme al D.Lgs. 36/2023?

Domanda: la specifica cosi' formulata e' ammissibile? Come va corretta? Cita
l'allegato II.5.
