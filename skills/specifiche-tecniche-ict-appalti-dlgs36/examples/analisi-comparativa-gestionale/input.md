# Input - acquisto di un gestionale: analisi comparativa preventiva

Un comune vuole **acquistare un software gestionale** per i servizi demografici.
L'ufficio propone di procedere direttamente all'acquisto di una **licenza d'uso di
un prodotto proprietario** gia' noto, senza altre valutazioni, perche' "e' quello
che usano tutti".

Domanda: prima di acquistare, quali passaggi impone il CAD? Quali soluzioni vanno
messe a confronto e con quali criteri? A quali condizioni e' legittimo scegliere il
software proprietario? Cita gli articoli.
