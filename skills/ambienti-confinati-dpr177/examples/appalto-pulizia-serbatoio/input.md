# Input - appalto per pulizia di un serbatoio (ambiente confinato)

Un'azienda alimentare deve far pulire internamente un grande serbatoio (ambiente
confinato) e affida i lavori a un'impresa esterna. Il responsabile chiede:

1. Che requisiti deve avere l'impresa appaltatrice per poter operare?
2. Cosa deve fare l'azienda committente prima e durante i lavori?
3. L'impresa puo' subappaltare parte dei lavori?
