# Input - classificazione dell'esposizione e obblighi

In un reparto di lavorazioni meccaniche, per una mansione il tecnico ha stimato
un'esposizione **LEX,8h = 86 dB(A)** con **ppeak = 132 dB(C)**.

Domande:
1. In quale fascia si colloca l'esposizione rispetto ai valori dell'art. 189?
2. Scatta l'obbligo di misurazione?
3. Quali obblighi (misure, segnaletica, DPI, sorveglianza) si attivano?
