# Input - sversamento con indagine sotto CSC

Durante lavori in un piazzale industriale si verifica uno **sversamento
accidentale** di gasolio. L'azienda mette in atto subito le misure di
contenimento. L'**indagine preliminare** sui parametri interessati mostra che le
concentrazioni nel suolo sono **inferiori alle CSC** dell'Allegato 5.

Domanda: cosa deve fare l'azienda, in quali tempi, e a chi va comunicato? Si apre
un procedimento di bonifica?
