# Nota

Esempio ancorato al **§3.5.2** (temperatura dell'aria esterna) delle NTC 2018, trascritto in
`references/fonti/ntc2018-par-3-5.md`.

Punti verificati sul testo (segni "−" verificati sull'immagine del PDF):

- **Periodo di ritorno 50 anni** per l'opera definitiva; TR ≥ 5/10 anni solo per fasi transitorie.
- **Zona I** (comprende la Lombardia): **Tmin = −15 − 4·as/1000** [3.5.1]; **Tmax = 42 − 6·as/1000** [3.5.2].
- Con as = 300 m: Tmin = −15 − 1,2 = **−16,2 °C**; Tmax = 42 − 1,8 = **40,2 °C**.

I calcoli sono applicazione diretta delle formule [3.5.1]/[3.5.2] (illustrazione). La **Fig. 3.5.1**
(zonazione) è un'immagine, non riprodotta: l'utente identifica la zona. La skill **non** calcola le temperature
superficiali né gli effetti.
