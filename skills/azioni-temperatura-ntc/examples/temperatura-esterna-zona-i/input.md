# Input - Temperatura dell'aria esterna in Zona I

Un progettista strutturale deve definire la **temperatura dell'aria esterna** (massima estiva e minima
invernale) per un edificio di nuova realizzazione (opera definitiva) situato in **Lombardia** (quindi **Zona
I**), a una quota **as = 300 m** sul livello del mare.

Chiede: quale **periodo di ritorno** si assume? Quali **valori di Tmax e Tmin** si ottengono con le formule
delle NTC 2018 §3.5.2?
