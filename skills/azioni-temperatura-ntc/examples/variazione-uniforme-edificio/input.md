# Input - Variazione termica uniforme di un edificio in c.a.

Un progettista deve inquadrare la **variazione termica uniforme** da assumere per un edificio con **struttura
in calcestruzzo armato esposta** (la temperatura non è azione fondamentale per la sicurezza). Vuole anche
stimare, a titolo illustrativo, la **dilatazione** libera di un elemento in c.a. di lunghezza **L = 10 m**.

Chiede: quale **ΔTu** si assume secondo le NTC 2018 §3.5.5 (Tab. 3.5.II)? Quale **coefficiente di dilatazione
termica** si usa (Tab. 3.5.III) e quanto vale la dilatazione libera dell'elemento?
