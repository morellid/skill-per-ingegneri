# Esempio — Input: classe della sezione di un elemento dissipativo (Tab. 7.5.I)

Struttura in acciaio a **telai** in zona sismica, progettata con **comportamento dissipativo**. Il progettista
adotta la **classe di duttilità CD "A"** con un **valore di base q0 = 5**.

Dati:

- **Travi dissipative**: profili laminati **IPE**, verificati come **sezioni di Classe 2** (secondo il §4.2.3.1).
- **Colonne primarie** del telaio in cui si prevede la formazione di zone dissipative: **NEd = 900 kN**;
  capacità a sforzo normale **Npl,Rd = 2500 kN**.

Domande:
1. La classe di sezione 2 delle travi dissipative è compatibile con CD "A" e q0 = 5 secondo la Tab. 7.5.I?
2. Le colonne primarie rispettano il limite sullo sforzo normale del §7.5.3?
