# Input - controlli AIA e cosa succede in caso di inosservanza

Un'installazione con AIA riceve un'**ispezione** dell'ARPA che rileva il **superamento di un valore
limite** prescritto. Ogni quanto avvengono le ispezioni ordinarie e quali **provvedimenti** può
adottare l'autorità competente in caso di **inosservanza** delle condizioni dell'AIA? Cita l'art.
29-decies del D.Lgs. 152/2006.
