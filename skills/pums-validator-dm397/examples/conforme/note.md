# Note sull'esempio CONFORME

Il caso e' costruito per illustrare un PUMS che soddisfa tutti i requisiti formali delle Linee guida italiane:
- Comune obbligato in via diretta (>100.000 ab.), non ricompreso in citta' metropolitana - quindi deve avere un PUMS proprio.
- Procedura in 8 passi documentata.
- 17 macro-obiettivi minimi + obiettivi specifici motivati.
- Indicatori con metodologia conforme al DM 396/2019 + Vademecum 2022.
- Monitoraggio biennale + aggiornamento quinquennale + valutazione pre-affidamento TPL.

L'output mostra il **formato di report atteso**: tabelle per ciascuna sotto-attivita', esito complessivo, rilievi categorizzati, avvertenza finale di rinvio al professionista.

Il caso include un unico rilievo non bloccante (cruscotto digitale assente) per dimostrare che anche un PUMS conforme puo' ricevere suggerimenti migliorativi.

I dati numerici (popolazione, budget) sono illustrativi.
