# Input - Ampliamento realizzato senza permesso di costruire

Un fabbricato è stato ampliato con un nuovo corpo di fabbrica (nuova volumetria) **senza permesso di
costruire**: si tratterebbe di un intervento che richiedeva il PdC, eseguito **in assenza di titolo**.
Il proprietario chiede se può ottenere una sanatoria e a quali condizioni; l'ampliamento rispetta le
NTC attuali ma la zona, all'epoca della costruzione, aveva un indice edificatorio inferiore.
