# Input - cambio destinazione d'uso da residenziale a turistico-ricettivo (Salva Casa)

## Richiesta dell'utente

> "Una mia cliente vuole trasformare il suo appartamento di 110 mq in un B&B con 5 camere (oltre la soglia per essere considerato extra-alberghiero professionale). L'unita' e' in zona B di Milano, fuori centro storico, niente vincoli. Non vogliamo fare nessuna opera edilizia: solo arredo, segnaletica interna, e qualche bagnetto in piu' ma con allacci esistenti. E' applicabile il Salva Casa? Quale modulo edilizio? E gli adempimenti SUAP turismo?"

## Inquadramento intervento

| Variabile | Valore |
|-----------|--------|
| Tipologia | Cambio destinazione d'uso senza opere edilizie sostanziali (art. 23-ter DPR 380) |
| Categoria attuale | a) Residenziale |
| Categoria nuova | a-bis) Turistico-ricettiva (B&B con 5 camere - oltre soglia LR Lombardia per gestione professionale) |
| Immobile | Esistente, costruzione 1995, stato legittimo accertato (DIA 1995 + SCIA manutenzione 2018) |
| Superficie utile (SU) | 110 mq |
| Parti strutturali coinvolte | NO |
| Frazionamento UI | NO |
| Modifica volumetria | NO |
| Modifica sagoma / prospetti | NO |
| Vincolo paesaggistico | NO |
| Vincolo culturale | NO |
| Vincolo idrogeologico | NO |
| Zona urbanistica | B (consolidato), fuori centro storico |
| Sismica | Zona 3 (Milano) - opere non strutturali |
| Cantiere | Mono-impresa per aggiunta bagno (durata < 30 gg, 2 lavoratori) |
| Attivita' antincendio | Da-verificare: B&B con 5 camere puo' rientrare in attivita' DPR 151/2011 punto 66 (strutture turistico-alberghiere) sopra 25 posti letto - in questo caso 5 camere x 2 = 10 posti letto, sotto soglia, ma verifica con VVF |
| Intervento energetico | NO |
| Modalita' di affidamento | Privato |
| Carico urbanistico | Da-valutare: aumento carico per nuova destinazione turistica |

## Domande dell'utente

1. Il Salva Casa si applica al cambio destinazione d'uso senza opere?
2. Quale modulo edilizio (CILA / SCIA / PdC)?
3. Quali allegati specifici per il cambio uso?
4. Gli adempimenti SUAP turismo (CIN, classificazione strutture extra-alberghiere) sono separati dal modulo edilizio?
