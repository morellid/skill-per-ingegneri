# Input - manutenzione straordinaria con frazionamento UI (CILA)

## Richiesta dell'utente

> "Un mio cliente vuole frazionare il suo appartamento di 92 mq in due unita' immobiliari distinte. Le opere previste sono tramezzi divisori in cartongesso, due nuovi bagni con allacci agli scarichi esistenti, due cucine con allacci a contatori esistenti. Non si tocca niente di strutturale. Siamo in zona B di un Comune toscano (Pistoia). Quale modulo devo presentare? Quali allegati?"

## Inquadramento intervento

| Variabile | Valore |
|-----------|--------|
| Tipologia | Manutenzione straordinaria (art. 3 co. 1 lett. b DPR 380) |
| Immobile | Esistente, anno di costruzione 1985, titolo: Concessione edilizia 1984 (stato legittimo accertato) |
| Superficie utile (SU) | 92 mq |
| Parti strutturali coinvolte | NO (solo tramezzi non portanti, impianti) |
| Frazionamento UI | SI (1 -> 2 UI) |
| Modifica volumetria | NO |
| Modifica sagoma / prospetti | NO |
| Cambio destinazione d'uso | NO (resta residenziale) |
| Vincolo paesaggistico | NO (zona non vincolata) |
| Vincolo culturale | NO |
| Vincolo idrogeologico | NO |
| Zona urbanistica | B (consolidato) |
| Sismica | Zona 3 (Comune di Pistoia) - opere non strutturali, no autorizzazione/deposito sismico |
| Cantiere | Unica impresa, durata stimata 30 gg, 3 lavoratori |
| Attivita' antincendio | NO |
| Intervento energetico | NO (solo divisori interni, no involucro) |
| Modalita' di affidamento | Privato (no appalto pubblico) |
