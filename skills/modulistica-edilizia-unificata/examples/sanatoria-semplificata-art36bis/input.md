# Input - sanatoria semplificata art. 36-bis (Salva Casa)

## Richiesta dell'utente

> "Un cliente sta vendendo un appartamento di 75 mq in un Comune lombardo (Brescia, zona B). In sede di pre-rogito notarile e' emersa una **parziale difformita'** rispetto al titolo: nel 2014, in attuazione di una **SCIA presentata nel 2013** per ristrutturazione interna (redistribuzione spazi e creazione di un secondo bagno), l'impresa ha eseguito le opere con una **diversa distribuzione dei tramezzi** rispetto agli elaborati di progetto: lo spostamento di un tramezzo divisorio non portante ha modificato la dimensione del bagno e dell'antibagno (la SU complessiva resta invariata, le UI restano una sola, non vi sono opere strutturali). Le opere sono state regolarmente eseguite ma non rispecchiano gli elaborati grafici asseverati. E' applicabile la sanatoria art. 36-bis Salva Casa?"

## Inquadramento intervento

| Variabile | Valore |
|-----------|--------|
| Tipologia opera realizzata | Spostamento di **tramezzo divisorio non portante** che ha modificato la dimensione del bagno e dell'antibagno rispetto al progetto, all'interno di una ristrutturazione interna (manutenzione straordinaria leggera ai sensi art. 3 co. 1 lett. b DPR 380) |
| Anno di realizzazione | 2014 (lavori eseguiti in attuazione di SCIA 2013) |
| Titolo originario all'epoca | **SCIA** ex art. 22 DPR 380 (regolarmente presentata nel 2013) |
| Tipologia di abuso | **Parziale difformita'** rispetto agli elaborati asseverati nella SCIA 2013 (regime art. 37 DPR 380) |
| Opere su parti strutturali | NO (tramezzo non portante) |
| Variazione SU complessiva | NO (la SU resta invariata, cambia solo la distribuzione interna) |
| Variazione numero UI | NO (resta una UI) |
| Variazione destinazione d'uso | NO (resta residenziale) |
| Variazione di sagoma/prospetti/volume | NO |
| Immobile | Esistente, costruzione 1992, stato legittimo accertato per la parte assentita (Concessione 1991 + SCIA 2013) |
| SU complessiva | 75 mq |
| Vincolo paesaggistico | NO |
| Vincolo culturale | NO |
| Vincolo idrogeologico | NO |
| Zona urbanistica | B (consolidato) |
| Sismica | Zona 3 (Brescia) - opere non strutturali |
| Conformita' urbanistica oggi (PGT Brescia 2024) | DA-VERIFICARE: la nuova distribuzione interna e' compatibile con NTA della zona B + scheda d'ambito? Indicatori principali: indici, distanze e RC non sono toccati (la difformita' e' interna). |
| Conformita' edilizia all'epoca (DPR 380 + RE Brescia 2014) | DA-VERIFICARE: il bagno di nuova dimensione rispetta i requisiti igienico-sanitari (DM 5/7/1975) e di aerazione vigenti nel 2014? |
| Condizione di tolleranza art. 34-bis co. 1-bis | Possibile valutazione: la "diversa realizzazione dei vani interni" e' espressamente tra le tolleranze esecutive del Salva Casa **purche'** non comporti variazione del numero di UI o della destinazione d'uso che modifichi la dotazione standard. Verificare se la difformita' interna rientra nella tolleranza esecutiva o se serve sanatoria. |

## Domande dell'utente

1. La parziale difformita' rientra nelle tolleranze esecutive art. 34-bis co. 1-bis o richiede sanatoria?
2. Se serve sanatoria, e' applicabile il Salva Casa? (Branch B - art. 36-bis via art. 37)
3. La doppia conformita' alleggerita e' verificabile nel caso specifico?
4. Quale modulo presentare (SCIA in sanatoria art. 36-bis)?
5. Quali allegati e quale documentazione probante?
6. Quale sanzione pecuniaria attesa?
7. Tempistica per chiudere la sanatoria prima del rogito?
