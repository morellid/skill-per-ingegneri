# Esempio — Input: capacity design e limite NEd/Npl,Rd di una colonna dissipativa

Struttura composta acciaio-calcestruzzo a **telai**, comportamento **dissipativo** (tipo b), in zona sismica. Si
valuta una **colonna primaria** di un telaio in cui si prevede la formazione di zone dissipative.

Dati:

- **Limite inferiore della capacità** della zona dissipativa (collegamento adiacente): **Epl,Rd = 800 kNm**.
- **Acciaio strutturale S355** → fattore di sovraresistenza del materiale **γov = 1,25** (§7.5.1).
- **Colonna primaria**: sforzo normale di progetto **NEd = 1800 kN**; capacità a sforzo normale
  **Npl,Rd = 5000 kN**.

Domande:
1. Quale limite superiore EU,Rd devo usare per dimensionare le componenti non dissipative?
2. La colonna primaria rispetta il limite sullo sforzo normale del §7.6.4?
