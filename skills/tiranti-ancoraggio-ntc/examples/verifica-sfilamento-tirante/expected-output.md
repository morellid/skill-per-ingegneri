# Output atteso - Verifica SLU a sfilamento di un tirante permanente (§6.6.2)

## 1. Impostazione della verifica

La verifica a **sfilamento** della fondazione dell'ancoraggio è la condizione [6.2.1] **Ed ≤ Rad**, da
soddisfare con la combinazione **A1+M1+R3** (coefficienti Tab. 6.2.I, 6.2.II e 6.6.I). La resistenza di
progetto è **Rad = Rak / γR**.

## 2. Resistenza caratteristica Rak (da prove di progetto, [6.6.1])

Con **3 ancoraggi di prova** (numero **> 2**), dalla **Tab. 6.6.II**: **ξa1 = 1,3** e **ξa2 = 1,2**. Quindi:

- Ra,m,medio / ξa1 = **780 / 1,3 = 600 kN**;
- Ra,m,min / ξa2 = **720 / 1,2 = 600 kN**;
- **Rak = Min{600 ; 600} = 600 kN** [6.6.1].

## 3. Resistenza di progetto Rad (Tab. 6.6.I)

L'ancoraggio è **permanente** → **γR = 1,2**:

- **Rad = Rak / γR = 600 / 1,2 = 500 kN**.

## 4. Esito della verifica

- **Ed = 450 kN ≤ Rad = 500 kN** → la verifica a sfilamento è **soddisfatta**.

(Se l'ancoraggio fosse **temporaneo**, sarebbe γR = 1,1 → Rad = 600 / 1,1 ≈ 545 kN.)

## Avvertenza

La skill **inquadra** la procedura (combinazione A1+M1+R3), i **fattori di correlazione ξ** (Tab. 6.6.II/III),
il **γR** (Tab. 6.6.I) e il confronto Ed ≤ Rad. I valori numerici sono un'**illustrazione** dell'applicazione
delle formule: la determinazione di Ra,m/Ra,c (da prove o calcolo), la scelta della combinazione e il
dimensionamento del tirante restano compito del progettista, che deve leggere il §6.6 delle NTC 2018.
