# Input - Verifica SLU a sfilamento di un tirante permanente

Un progettista geotecnico deve inquadrare la **verifica a sfilamento** di un **tirante di ancoraggio
permanente** di una paratia. La resistenza è determinata da **prove di progetto** eseguite su **3 ancoraggi di
prova**, che hanno dato: **Ra,m medio = 780 kN** e **Ra,m minimo = 720 kN**. La massima azione di progetto sul
tirante è **Ed = 450 kN**.

Chiede: come si determina la **resistenza di progetto Rad** secondo le NTC 2018 §6.6.2? Quali **fattori di
correlazione** e quale **coefficiente parziale γR** si usano? La verifica è soddisfatta?
