# Nota

Esempio ancorato al **§6.6.2** (verifica SLU a sfilamento) delle NTC 2018, trascritto in
`references/fonti/ntc2018-par-6-6.md`.

Punti verificati sul testo:

- Condizione **[6.2.1] Ed ≤ Rad**, combinazione **A1+M1+R3**; **Rad = Rak / γR**.
- **Tab. 6.6.I**: γR = **1,1** temporanei, **1,2** permanenti.
- **[6.6.1]** (da prove): **Rak = Min{ Ra,m,medio / ξa1 ; Ra,m,min / ξa2 }**; **Tab. 6.6.II** con 3 ancoraggi
  di prova (>2) → ξa1 = 1,3, ξa2 = 1,2.
- Calcolo del caso: 780/1,3 = 600; 720/1,2 = 600 → Rak = 600 kN; Rad = 600/1,2 = **500 kN** (permanente);
  Ed = 450 ≤ 500 → verificato.

I calcoli sono applicazione diretta delle formule/tabelle NTC (illustrazione). La skill **non** determina
Ra,m/Ra,c né dimensiona il tirante: la resistenza va confermata sempre con prove in sito (§6.6.1).
