# Input - Appalto integrato (progettazione + esecuzione)

Una stazione appaltante affida con **appalto integrato** (progettazione esecutiva + esecuzione) un
intervento partendo dal **progetto di fattibilità tecnico-economica (PFTE)**. Il RUP chiede: quando va
verificato il PFTE e quando il progetto esecutivo? Chi può fare la verifica e cosa non può fare
contemporaneamente?
