# Input - Limite di drift per telaio in c.a. con tamponature fragili (CU II)

Un progettista deve verificare la deformabilità allo SLD di un edificio residenziale (**classe d'uso II**), con
telaio in c.a. e **tamponature in laterizio (fragili) collegate rigidamente** alla struttura. L'altezza di piano
è h = 3,00 m.

Chiede secondo le NTC 2018 §7.3.6.1:
- A quale **stato limite** va riferita la verifica di rigidezza?
- Qual è il **limite di spostamento di interpiano** e come si esprime la verifica?
