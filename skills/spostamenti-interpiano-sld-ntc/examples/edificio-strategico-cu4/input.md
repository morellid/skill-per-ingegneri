# Input - Edificio strategico (CU IV) con tamponature duttili

Un progettista deve verificare la deformabilità di un edificio strategico (**classe d'uso IV**, es. sede di
protezione civile) con **tamponature duttili collegate rigidamente** alla struttura, altezza di piano h = 3,50 m.

Chiede secondo le NTC 2018 §7.3.6.1:
- A quale **stato limite** va riferita la verifica per una CU IV?
- Qual è il **limite di spostamento di interpiano** applicabile?
