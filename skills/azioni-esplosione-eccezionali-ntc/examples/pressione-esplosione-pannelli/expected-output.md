# Output atteso - Pressione statica equivalente (Categoria 2) (§3.6.2.3, [3.6.5])

## 1. Ammissibilità

- **Rapporto Av/V = 0,10 m⁻¹**: rientra nel vincolo **0,05 m⁻¹ ≤ Av/V ≤ 0,15 m⁻¹** [3.6.6] → ammissibile.
- **Volume V = 500 m³ ≤ 1.000 m³**: le espressioni [3.6.5a]/[3.6.5b] sono applicabili.

## 2. Pressione statica equivalente nominale

La pressione di progetto pd è il **valore maggiore** fra [3.6.5a] e [3.6.5b] (con pv = 3 kN/m²):

- **[3.6.5a]**: pd = 3 + pv = 3 + 3 = **6,0 kN/m²**;
- **[3.6.5b]**: pd = 3 + pv/2 + 0,04/(Av/V)² = 3 + 3/2 + 0,04/(0,10)² = 3 + 1,5 + 0,04/0,01 = 3 + 1,5 + 4 =
  **8,5 kN/m²**.

Quindi **pd = max{6,0 ; 8,5} = 8,5 kN/m²**, intesa agente **simultaneamente su tutte le pareti** dell'ambiente.

## 3. Elementi chiave

Indipendentemente dal calcolo sopra, **tutti gli elementi chiave e le loro connessioni** vanno progettati per
una pressione statica equivalente **pd = 20 kN/m²** applicata **da ogni direzione**, insieme alle reazioni
trasmesse dagli elementi collegati soggetti alla stessa pressione.

## Avvertenza

La skill **inquadra** le formule [3.6.5a]/[3.6.5b], il vincolo [3.6.6] e i limiti di validità. Il valore pd =
8,5 kN/m² è un'**illustrazione**: la verifica delle pareti/elementi, la combinazione eccezionale e la
progettazione dei pannelli di sfogo restano compito del progettista, secondo il §3.6.2 delle NTC 2018.
