# Nota

Esempio ancorato al **§3.6.2.3** (modellazione delle azioni da esplosione, Categoria 2) delle NTC 2018,
trascritto in `references/fonti/ntc2018-par-3-6-2.md`.

Punti verificati sul testo (operatori "+" verificati sull'immagine del PDF):

- **[3.6.5a]**: **pd = 3 + pv**; **[3.6.5b]**: **pd = 3 + pv/2 + 0,04/(Av/V)²**; si assume il **maggiore**.
- **[3.6.6]**: **0,05 m⁻¹ ≤ Av/V ≤ 0,15 m⁻¹**; espressioni valide per **V ≤ 1.000 m³**.
- **Elementi chiave**: **pd = 20 kN/m²** da ogni direzione.
- Con pv = 3, Av/V = 0,10, V = 500 m³: [3.6.5a] = 6,0; [3.6.5b] = 3 + 1,5 + 0,04/0,01 = 8,5 → pd = **8,5
  kN/m²**.

I calcoli sono applicazione diretta delle formule (illustrazione). La skill **non** verifica le pareti né
progetta i pannelli di sfogo.
