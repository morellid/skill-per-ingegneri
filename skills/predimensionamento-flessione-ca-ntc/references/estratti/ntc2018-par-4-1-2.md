> del 20/02/2018 - Suppl. Ord. n. 8 (fonte pubblica, SHA256 verificato).

## par. 4.1.2.1.1.1 - Resistenza di progetto a compressione del calcestruzzo

Formula [4.1.3] del NTC 2018:

```
fcd = alpha_cc * fck / gamma_c           (eq. [4.1.3], par. 4.1.2.1.1.1)
```

dove (valori verificati nel testo PDF, par. 4.1.2.1.1.1):
- `alpha_cc = 0,85` coefficiente riduttivo per resistenze di lunga durata
- `fck` resistenza caratteristica cilindrica del calcestruzzo a 28 giorni (MPa)
- `gamma_c = 1,5` coefficiente parziale del calcestruzzo per SLU ordinario

**Nota errore PR #89**: la skill originale citava "eq. 4.1.4" per fcd. La formula fcd
e' l'**eq. [4.1.3]** del NTC 2018. L'eq. [4.1.4] riguarda la resistenza a trazione
fctd = fctk / gamma_c. Questo estratto riporta la numerazione corretta letta dal PDF.

## par. 4.1.2.1.1.3 - Resistenza di progetto dell'acciaio

Formula [4.1.5] del NTC 2018:

```
fyd = fyk / gamma_s                      (eq. [4.1.5], par. 4.1.2.1.1.3)
```

dove (valori verificati nel testo PDF, par. 4.1.2.1.1.3):
- `fyk` tensione caratteristica di snervamento dell'acciaio ordinario (par. 11.3.2)
- `gamma_s = 1,15` coefficiente parziale dell'acciaio per tutti i tipi (ordinario
  e da precompressione), per ogni combinazione SLU

Modulo elastico acciaio (valore standard EN 1992-1-1, par. 11.3.2.1 NTC):
```
Es = 200000 MPa
eps_yd = fyd / Es
```

## par. 4.1.2.1.2.1 - Modello stress-block rettangolare equivalente (Classe 1, fck <= 50 MPa)

Il NTC 2018 par. 4.1.2.1.2.1 descrive tre modelli sigma-eps per il calcestruzzo
(Fig. 4.1.1): (a) parabola-rettangolo, (b) triangolo-rettangolo, (c) rettangolo
(stress block). Il modello (c) e' il piu' semplice e viene usato da questa skill.

Per fck <= 50 MPa (Classe 1, da C8/10 a C50/60), il testo NTC specifica:

```
eps_cu = 0,0035   (= 0,35%)   deformazione ultima di progetto        (par. 4.1.2.1.2.1)
```

I parametri del modello stress-block rettangolare (Fig. 4.1.1(c), EN 1992-1-1 par. 3.1.7):

```
lambda = 0,8    profondita' equivalente del blocco rettangolare
eta    = 1,0    intensita' della tensione nel blocco (eta * fcd)
```

Per fck > 50 MPa (Classe 2), il NTC specifica che "purche' si adottino opportune
limitazioni quando si usa il modello (c)": i parametri lambda ed eta si riducono
(dalla formula EN 1992-1-1 par. 3.1.7(3)). **La skill v0.1 copre solo Classe 1**.

## par. 4.1.2.3.4.1 - Ipotesi di base

Ipotesi adottate nel calcolo (par. 4.1.2.3.4.1, lette dal PDF):
- conservazione delle sezioni piane (Bernoulli-Navier)
- perfetta aderenza tra acciaio e calcestruzzo
- resistenza a trazione del calcestruzzo nulla

## par. 4.1.2.3.4.2 - Verifica di resistenza SLU in flessione semplice

Il NTC 2018 par. 4.1.2.3.4.2 definisce la verifica come MRd >= MEd (eq. [4.1.18a]).
La formula chiusa per MRd di sezione rettangolare singolarmente armata si ricava
applicando le ipotesi di par. 4.1.2.3.4.1 con il modello stress-block:

### Equilibrio (Classe 1, eta = 1, lambda = 0,8):

```
Cc = T
0,8 * b * x * fcd = As * fyd
=> x = (As * fyd) / (0,8 * b * fcd)           (profondita' asse neutro)
```

### Compatibilita' (sezioni piane):

```
eps_s = eps_cu * (d - x) / x                  (deformazione acciaio teso)
```

- Se `eps_s >= eps_yd`: acciaio snerva, sigma_s = fyd, formula valida.
- Se `eps_s < eps_yd`: sezione **sovra-armata**, la formula MRd = As * fyd * z
  **sovrastima** la resistenza. La skill rifiuta il calcolo (ValueError).

### Momento resistente:

```
y_c = (lambda * x) / 2 = 0,4 * x              (centroide blocco rettangolare)
z   = d - y_c = d - 0,4 * x                   (braccio della coppia interna)
MRd = As * fyd * z = As * fyd * (d - 0,4 * x)
```

## par. 4.1.1.1 - Limite x/d per ridistribuzione momenti (travi continue)

Il NTC 2018 par. 4.1.1.1 (analisi elastica lineare) fissa il limite:

```
x/d <= 0,45    per fck <= 50 MPa   (eq. [4.1.1], ridistribuzione momenti)
x/d <= 0,35    per fck > 50 MPa   (eq. [4.1.2], ridistribuzione momenti)
```

**CONTESTO**: questo limite si applica alle sezioni critiche di travi continue e
solette nelle quali si effettua la ridistribuzione dei momenti flettenti (par. 4.1.1.1).
NON e' un limite universale per la verifica di resistenza di singola sezione in
flessione semplice. Per flessione semplice non sismica, la sola condizione necessaria
e' che l'acciaio snervi (eps_s >= eps_yd). La skill produce un'avvertenza quando
x/d > 0,45 con questo contesto preciso, senza bloccare il calcolo.

## Note sull'uso da parte della skill

- `alpha_cc = 0,85` default, sovrascrivibile per casi speciali
- `gamma_c = 1,5` default SLU ordinario
- `gamma_s = 1,15` default tutti i tipi di acciaio
- `Es = 200000 MPa` default (valore standard EN 1992-1-1 / NTC par. 11.3.2.1)
- `eps_cu = 0,0035` default (Classe 1, fck <= 50 MPa)
- `lambda = 0,8`, `eta = 1` fissi (Classe 1 only)
- Input utente: `b` (mm), `h` (mm), `d` (mm), `As` (mm^2), `fck` (MPa), `fyk` (MPa)
- Output: `fcd`, `fyd`, `eps_yd`, `x` (mm), `x/d`, `eps_s`, `z` (mm), `MRd` (Nmm e kNm)
