# Estratto - NTC 2018 par. 4.1.2 (Costruzioni di calcestruzzo)

> Sintesi strutturata dei contenuti rilevanti del par. 4.1.2 del DM 17/01/2018 (NTC 2018), pubblicato in GU n. 42 del 20/02/2018 - Suppl. Ord. n. 8 (fonte pubblica). Le formule e i parametri sono riportati per uso operativo della skill; il testo completo di NTC e' nella fonte ufficiale.

## par. 4.1.2.1.1 - Resistenze di progetto

### Calcestruzzo

```
f_cd = alpha_cc * fck / gamma_c           (eq. 4.1.4)
```

dove:
- `alpha_cc = 0.85` coefficiente riduttivo per resistenza di lunga durata (carichi di lunga durata)
- `fck` resistenza caratteristica cilindrica del calcestruzzo (MPa)
- `gamma_c = 1.5` coefficiente parziale del calcestruzzo per Stato Limite Ultimo PERS (Tab. 4.1.II)

Per combinazioni eccezionali (sismiche tipo C4) o per provvedimenti di rinforzo strutturale, NTC ammette `gamma_c = 1.0` con specifica documentazione.

### Acciaio per c.a.

```
f_yd = fyk / gamma_s                      (par. 4.1.2.1.1.3)
```

dove:
- `fyk` tensione caratteristica di snervamento (MPa). Per B450C e B450A: fyk = 450 MPa (NTC par. 11.3.2.1).
- `gamma_s = 1.15` coefficiente parziale dell'acciaio per SLU PERS (Tab. 4.1.II)

```
Es = 200000 MPa                           (par. 4.1.2.1.1.2)
eps_yd = f_yd / Es
```

Per B450C: `f_yd = 391.30 MPa`, `eps_yd = 0.001957`.

## par. 4.1.2.1.1 - Diagrammi tensione-deformazione

### Diagramma stress-block rettangolare equivalente (calcestruzzo SLU)

Per calcestruzzo di **Classe 1** (`fck <= 50 MPa`):

```
lambda = 0.8                              (profondita' del blocco rettangolare)
eta    = 1.0                              (tensione costante eta * f_cd)
eps_cu = 0.0035                           (deformazione ultima)
```

Per **Classe 2** (`fck > 50 MPa`) NTC eq. 4.1.10-4.1.11:

```
lambda = 0.8 - (fck - 50) / 400           (lambda < 0.8)
eta    = 1.0 - (fck - 50) / 200           (eta < 1.0)
eps_cu = 0.0026 + 0.035 * [(90 - fck)/100]^4  (decrescente)
```

**La skill v0.1 copre solo Classe 1**. Per fck > 50 MPa il modulo solleva ValueError.

### Diagramma elasto-plastico perfetto (acciaio SLU)

```
sigma_s = Es * eps_s    per |eps_s| <= eps_yd       (zona elastica)
sigma_s = sgn(eps_s) * f_yd  per |eps_s| > eps_yd   (zona plastica)
```

(NTC par. 4.1.2.1.1.2). La skill assume sigma_s = f_yd quando l'acciaio snerva (eps_s >= eps_yd).

## par. 4.1.2.3.4.2 - Flessione semplice SLU

Per sezione rettangolare `b x h` con armatura tesa A_s a quota `d`:

### Equilibrio (singolarmente armata, acciaio snervato)

```
Cc = T
lambda * b * x * eta * f_cd = A_s * f_yd
```

per Classe 1 (eta = 1, lambda = 0.8) si semplifica:

```
0.8 * b * x * f_cd = A_s * f_yd
->  x = (A_s * f_yd) / (0.8 * b * f_cd)         (profondita' asse neutro)
```

### Compatibilita' (sezione piana)

```
eps_s = eps_cu * (d - x) / x
```

- Se `eps_s >= eps_yd`: acciaio snerva, formula valida.
- Se `eps_s < eps_yd`: sezione **sovra-armata**, l'acciaio non snerva, la formula M_Rd = A_s f_yd z **sovrastima** la resistenza. **La skill rifiuta il calcolo** e segnala l'errore: il progettista deve aumentare b/h, ridurre A_s, o passare a calcolo iterativo con sigma_s elastico (fuori scope skill v0.1).

### Momento resistente

Posizione della risultante di compressione (centroide blocco rettangolare):

```
y_c = (lambda * x) / 2 = 0.4 * x        (per lambda = 0.8)
z   = d - y_c = d - 0.4 * x             (braccio della coppia interna)
```

```
M_Rd = A_s * f_yd * z = A_s * f_yd * (d - 0.4 * x)
```

### Limite di duttilita'

NTC non fissa un limite tassativo su `x/d` per flessione semplice non sismica (basta che l'acciaio snerva), ma la **Circolare 7/2019 par. C4.1.2** raccomanda `x/d <= 0.45` per garantire duttilita' adeguata. In zona sismica con `q > 1.5` (CD"A" / CD"B") si applicano i limiti piu' stringenti di NTC par. 7.4.4 e 7.4.6 (rho_min, rho_max, gerarchia delle resistenze) - **non coperti dalla skill**.

La skill produce un'avvertenza quando x/d > 0.45 ma non blocca il calcolo, perche' la formula resta valida finche' l'acciaio snerva.

## Note sull'uso da parte della skill

- `alpha_cc = 0.85` default (sovrascrivibile per carichi di breve durata)
- `gamma_c = 1.5` default (sovrascrivibile per combinazioni eccezionali)
- `gamma_s = 1.15` default
- `Es = 200000 MPa` default
- `eps_cu = 0.0035` default (Classe 1)
- `lambda = 0.8` e `eta = 1` fissi (Classe 1 only)
- Input utente: `b` (mm), `h` (mm), `d` (mm), `A_s` (mm^2), `fck` (MPa), `fyk` (MPa)
- Output: `f_cd`, `f_yd`, `eps_yd`, `x` (mm), `x/d`, `eps_s`, `z` (mm), `M_Rd` (Nmm e kNm), flag duttilita'
