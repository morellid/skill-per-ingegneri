# Input - Tipi di consolidamento e cautele

Un progettista deve consolidare le fondazioni di un **edificio esistente** che manifesta cedimenti differenziali,
secondo le NTC 2018 §6.10.

Chiede:
- Quali sono i **tipi (metodi) di consolidamento geotecnico** previsti dalla norma per una struttura esistente?
- Ci sono interventi che richiedono **particolari cautele**? Quali e perché?
