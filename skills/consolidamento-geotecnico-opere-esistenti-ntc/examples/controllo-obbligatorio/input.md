# Input - Quando il controllo dell'efficacia è obbligatorio

Un progettista programma un intervento di consolidamento (iniezioni nel terreno di fondazione) su un edificio
esistente, secondo le NTC 2018 §6.10.

Chiede:
- Il **controllo dell'efficacia** del consolidamento è sempre obbligatorio? In quale caso lo diventa?
- Come vanno impostati il **progetto** (rapporto con la parte strutturale) e il **monitoraggio**?
