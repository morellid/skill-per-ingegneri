# Input - accordo in mediazione: è esecutivo?

In una mediazione le parti hanno **raggiunto un accordo**. Vorrei sapere: come si formalizza (verbale
e allegati), e l'accordo è **direttamente esecutivo** o serve un passaggio dal giudice? Cambia
qualcosa se le parti **sono assistite dagli avvocati** o no? Cita il D.Lgs. 28/2010.
