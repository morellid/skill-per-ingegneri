# Esempio - input (caso conforme)

## Descrizione attrezzatura

Serbatoio di accumulo aria compressa per uso pneumatico in ambito officina meccanica:

- Tipo: recipiente cilindrico verticale.
- Pressione massima ammissibile **PS** = 11 bar (relativi).
- Volume proprio **V** = 100 litri.
- Fluido contenuto: aria compressa.
- Temperatura massima ammissibile **TS** = 60 gradi C; temperatura minima ammissibile -10 gradi C.
- Scheda di sicurezza del fluido: aria atmosferica, **non e' classificata** secondo nessuna delle classi di pericolo dell'art. 9 c. 1 lett. a) del D.Lgs 26/2016 (non infiammabile, non comburente in concentrazione superiore a O2 atmosferico, non tossica, non instabile).
- Stato fisico operativo: gas/vapore (gas compresso).
- Destinazione: alimentazione utensili pneumatici in officina meccanica fissa.
- Produzione: serie industriale standard (non pezzo unico).
- Data prevista di immissione sul mercato: 2026-09-01.
- Fabbricante: societa' italiana, marchio proprio.
