# Nota

Esempio ancorato al **§5.1.3.3** (carichi verticali da traffico) delle NTC 2018, trascritto in
`references/fonti/ntc2018-par-5-1.md`.

Punti verificati sul testo:

- **Tab. 5.1.I**: per **6,0 m ≤ w**, **nl = Int(w/3)**, corsia **3,00 m**, zona rimanente **w − 3,00·nl**.
  Con w = 10,50 m → nl = Int(3,5) = **3**; zona rimanente 1,50 m.
- **Schemi di Carico 1-5** (§5.1.3.3.3): Schema 1 (tandem + distribuito, verifiche globali e locali), Schema
  2 (asse singolo, locali; ruota 200 kN), Schema 3 (150 kN), Schema 4 (10 kN), Schema 5 (folla 5,0 kN/m²,
  combinazione 2,5 kN/m²). Schemi 6.a/b/c solo per luce > 300 m.
- **Tab. 5.1.II**: Qik/qik = corsia 1: 300 / 9,00; corsia 2: 200 / 2,50; corsia 3: 100 / 2,50; altre/zona
  rimanente: 0 / 2,50.

Il conteggio `Int(w/3)` e l'assegnazione delle intensità sono applicazione diretta delle tabelle NTC. La
skill **non** individua la disposizione più gravosa dei carichi né calcola le sollecitazioni; le **Fig.
5.1.1/5.1.2** (numerazione corsie, geometrie/interassi degli Schemi) sono figure, non riprodotte → norma/EC1.
