# Input - cronoprogramma screening progetto di competenza regionale

Un proponente deve avviare la verifica di assoggettabilita' a VIA per un impianto
che rientra tra i progetti dell'allegato IV alla Parte Seconda del D.Lgs.
152/2006 (competenza regionale). Chiede:

1. Quali sono i termini del procedimento di verifica di assoggettabilita' (art.
   19) e quale documento deve presentare?
2. La regione puo' aver modificato le soglie dimensionali? In che modo?
