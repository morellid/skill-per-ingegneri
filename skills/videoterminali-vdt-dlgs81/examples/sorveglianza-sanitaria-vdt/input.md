# Input - sorveglianza sanitaria del videoterminalista

Per un videoterminalista (uso del VDT ben oltre le 20 ore settimanali):
- **Lavoratore A**: 45 anni, giudicato idoneo **senza** prescrizioni.
- **Lavoratore B**: 52 anni.

Domande:
1. A quali rischi è riferita la sorveglianza sanitaria (art. 176)?
2. Con quale periodicità vanno le visite per A e per B?
3. Chi paga eventuali occhiali "da videoterminale"?
