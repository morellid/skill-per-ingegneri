# Input - qualifica di videoterminalista e pause

Un impiegato amministrativo lavora al computer (con schermo, tastiera e mouse) per circa
**30 ore a settimana** in modo abituale. Il contratto collettivo aziendale **non**
disciplina le interruzioni per l'uso del videoterminale.

Domande:
1. È un "videoterminalista" ai sensi dell'art. 173?
2. Che pause gli spettano (art. 175)?
3. Le pause si possono spostare tutte a fine giornata per uscire prima?
