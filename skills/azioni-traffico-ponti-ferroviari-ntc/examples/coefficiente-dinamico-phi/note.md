# Nota

Esempio ancorato al **NTC 2018 §5.2.2.2.3** (effetti dinamici), trascritto in
`references/fonti/ntc2018-par-5-2.md`.

Formule verificate sull'immagine (pagina PDF 171, GU 167), perché `pdftotext` altera la radice e i pedici:

- **Φ2 = 1,44 / (√L_φ − 0,2) + 0,82**, con **1,00 ≤ Φ2 ≤ 1,67** [5.2.6];
- **Φ3 = 2,16 / (√L_φ − 0,2) + 0,73**, con **1,00 ≤ Φ3 ≤ 2,00** [5.2.7].

Con **L_φ = 20 m** (trave semplicemente appoggiata: L_φ = luce, Tab. 5.2.II): √20 = 4,472; √20 − 0,2 = 4,272;
Φ2 = 1,44/4,272 + 0,82 ≈ **1,16**; Φ3 = 2,16/4,272 + 0,73 ≈ **1,24**. Sono semplici applicazioni aritmetiche
della formula normativa. La skill inquadra formula, limiti e campo di applicazione; **non** esegue l'analisi
dinamica con treni reali né le verifiche.
