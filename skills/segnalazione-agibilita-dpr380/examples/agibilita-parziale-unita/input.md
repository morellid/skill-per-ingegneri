# Input - agibilità parziale di una singola unità immobiliare

In un fabbricato plurifamiliare **non ancora ultimato in tutte le unità**, una **singola unità
immobiliare** è completata. Posso chiederne l'**agibilità parziale**? A quali condizioni? E se un
locale ha **altezza 2,50 m**, è possibile? Cita l'art. 24 del D.P.R. 380/2001.
