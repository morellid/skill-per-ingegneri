# Input - Contenuto della DoP e caratteristiche non dichiarate

Un fabbricante di **aggregati** per calcestruzzo deve immettere sul mercato un prodotto coperto da una
**norma armonizzata** (hEN). Vuole redigere la **Dichiarazione di Prestazione (DoP)**. Per l'uso previsto
dichiara la prestazione di alcune caratteristiche essenziali (es. granulometria, forma dei granuli), ma per
un'altra caratteristica essenziale prevista dalla norma (es. rilascio di sostanze pericolose) **non intende
dichiarare** una prestazione.

Chiede: la DoP è obbligatoria? Quali informazioni deve contenere secondo il Reg. UE 305/2011? Come si tratta
la caratteristica essenziale per la quale non dichiara la prestazione?
