# Estratto - NTC 2018 par. 3.4 (Azioni della neve)

> Sintesi strutturata dei contenuti del par. 3.4 del DM 17/01/2018 (NTC 2018), pubblicato in GU n. 42 del 20/02/2018 - Suppl. Ord. n. 8 (fonte pubblica). Le formule e i parametri zonali sono riportati per uso operativo della skill; il testo completo di NTC e' nella fonte ufficiale.

## par. 3.4.1 - Carico neve sulla copertura

Il carico neve sulla copertura per metro quadrato di proiezione orizzontale si calcola con:

```
q_s = mu_1 * q_sk * c_E * c_t      (eq. 3.4.1)
```

dove:
- `q_sk` valore caratteristico del carico neve al suolo (par. 3.4.2)
- `mu_1` coefficiente di forma della copertura (par. 3.4.5)
- `c_E` coefficiente di esposizione (par. 3.4.3, Tab. 3.4.I)
- `c_t` coefficiente termico (par. 3.4.4)

Tutti i carichi sono in kN/m^2.

## par. 3.4.2 - Valore caratteristico del carico neve al suolo q_sk

Il territorio italiano e' suddiviso in 4 zone:

- **Zona I-Alpina**: Aosta, Belluno, Bergamo, Biella, Bolzano, Brescia, Como, Cuneo, Lecco, Pordenone, Sondrio, Torino, Trento, Udine, Verbania, Vercelli, Vicenza
- **Zona I-Mediterranea**: Alessandria, Ancona, Asti, Bologna, Cremona, Forli'-Cesena, Lodi, Milano, Modena, Monza-Brianza, Novara, Padova, Parma, Pavia, Pesaro-Urbino, Piacenza, Pistoia, Ravenna, Reggio Emilia, Rimini, Rovigo, Treviso, Varese, Venezia, Verona
- **Zona II**: Arezzo, Ascoli Piceno, Bari, Barletta-Andria-Trani, Campobasso, Chieti, Fermo, Ferrara, Firenze, Foggia, Frosinone, Genova, Gorizia, Imperia, Isernia, La Spezia, L'Aquila, Lucca, Macerata, Mantova, Massa-Carrara, Perugia, Pescara, Prato, Rieti, Savona, Teramo, Terni, Trieste
- **Zona III**: Agrigento, Avellino, Benevento, Brindisi, Cagliari, Caltanissetta, Carbonia-Iglesias, Caserta, Catania, Catanzaro, Cosenza, Crotone, Enna, Grosseto, Latina, Lecce, Livorno, Matera, Medio Campidano, Messina, Napoli, Nuoro, Ogliastra, Olbia-Tempio, Oristano, Palermo, Pisa, Potenza, Ragusa, Reggio Calabria, Roma, Salerno, Sassari, Siena, Siracusa, Taranto, Trapani, Vibo Valentia, Viterbo

(Elenchi indicativi - il professionista deve verificare la zona del Comune sulla copia ufficiale di NTC).

Le formule per `q_sk` (kN/m^2) sono le seguenti:

**Zona I-Alpina**:
```
q_sk = 1.50                                  per a_s <= 200 m
q_sk = 1.39 * [1 + (a_s/728)^2]              per a_s >  200 m   (eq. 3.4.1 par. 3.4.2)
```

**Zona I-Mediterranea**:
```
q_sk = 1.50                                  per a_s <= 200 m
q_sk = 1.35 * [1 + (a_s/602)^2]              per a_s >  200 m   (eq. 3.4.2 par. 3.4.2)
```

**Zona II**:
```
q_sk = 1.00                                  per a_s <= 200 m
q_sk = 0.85 * [1 + (a_s/481)^2]              per a_s >  200 m   (eq. 3.4.3 par. 3.4.2)
```

**Zona III**:
```
q_sk = 0.60                                  per a_s <= 200 m
q_sk = 0.51 * [1 + (a_s/481)^2]              per a_s >  200 m   (eq. 3.4.4 par. 3.4.2)
```

`a_s` e' l'altitudine del sito sul livello del mare in m. La skill incorpora le 4 zone e le formule come definite nel par. 3.4.2.

## par. 3.4.3 - Coefficiente di esposizione c_E (Tab. 3.4.I)

| Topografia          | c_E |
|---------------------|-----|
| Battuta dai venti   | 0.9 |
| Normale             | 1.0 |
| Riparata            | 1.1 |

Default consigliato: `normale`. La classificazione "battuta dai venti" si applica a coperture molto esposte; "riparata" a coperture in zone protette da terreno rilevante o costruzioni alte. La skill richiede la classe come input.

## par. 3.4.4 - Coefficiente termico c_t

Tiene conto della riduzione del carico per scioglimento causato dalla trasmissione di calore attraverso la copertura. In assenza di studi specifici si pone `c_t = 1.0` (default skill).

## par. 3.4.5.2.1 - Coefficiente di forma mu_1 per coperture ad una/due falde

```
mu_1 = 0.8                          per  0 <= alpha <= 30 deg
mu_1 = 0.8 * (60 - alpha) / 30      per 30 < alpha <  60 deg
mu_1 = 0                            per alpha >= 60 deg
```

`alpha` e' l'inclinazione della falda in gradi. La formula vale per coperture regolari ad una o due falde.

## par. 3.4.5.5 - Accumuli, deriva, scorrimento (NON coperti dalla skill)

NTC e Circolare C3.4 prevedono coefficienti aggiuntivi `mu_2`, `mu_3` e situazioni di accumulo / deriva / scorrimento per coperture multiple, sporgenze, parapetti, ostruzioni. **La skill copre solo il caso ordinario `mu_1`**: per casi non ordinari va eseguito un calcolo separato o consultata letteratura specialistica.

## Note sull'uso da parte della skill

- Zona neve: input utente (stringa I-Alpina / I-Mediterranea / II / III)
- `a_s`: input utente (m s.l.m.)
- `alpha`: input utente (gradi della falda)
- Classe esposizione `c_E`: input utente (battuta_dai_venti / normale / riparata)
- `c_t`: default 1.0
- Output: `q_sk`, `mu_1`, `c_E`, `q_s` (in kN/m^2)
