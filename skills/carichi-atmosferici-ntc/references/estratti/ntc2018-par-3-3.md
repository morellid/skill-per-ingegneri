# Estratto - NTC 2018 par. 3.3 (Azioni del vento)

> Sintesi strutturata dei contenuti del par. 3.3 del DM 17/01/2018 (NTC 2018), pubblicato in GU n. 42 del 20/02/2018 - Suppl. Ord. n. 8 (fonte pubblica). Le formule e i parametri sono riportati per uso operativo della skill; il testo completo di NTC e' nella fonte ufficiale.

## par. 3.3.1 - Pressione del vento

La pressione esercitata dal vento sull'unita' di superficie di un elemento costruttivo si calcola con:

```
p = q_r * c_e * c_p * c_d           (eq. 3.3.1)
```

dove:
- `q_r` pressione cinetica di riferimento (par. 3.3.4)
- `c_e` coefficiente di esposizione (par. 3.3.7)
- `c_p` coefficiente di forma (o aerodinamico, par. 3.3.8)
- `c_d` coefficiente dinamico (par. 3.3.9)

## par. 3.3.2 - Velocita' di riferimento del vento

La velocita' di riferimento `v_b` (m/s) e':

```
v_b = v_b_0                                       per a_s <= a_0
v_b = v_b_0 * (1 + k_s * (a_s/a_0 - 1))           per a_0 < a_s <= 1500 m   (eq. 3.3.2)
```

dove:
- `v_b_0`, `a_0`, `k_s` sono parametri della Tab. 3.3.I in funzione della zona del territorio italiano (1-9). La Tab. 3.3.I dipende dalla regione/provincia/zona e va consultata sulla copia ufficiale di NTC: la skill richiede i tre parametri come input dell'utente
- `a_s` altitudine del sito sul livello del mare in m

Per `a_s > 1500 m` la norma rinvia a indagini specifiche; la skill rifiuta il calcolo.

## par. 3.3.3-3.3.4 - Coefficiente di ritorno e pressione cinetica di riferimento

Per periodi di ritorno `T_R` diversi da 50 anni (vita di riferimento standard):

```
c_r = 0.75 * sqrt(1 - 0.2 * ln(-ln(1 - 1/T_R)))    (eq. 3.3.3)
v_r = c_r * v_b
q_r = 0.5 * rho * v_r^2                              (eq. 3.3.4)
```

con `rho = 1.25 kg/m^3` (densita' aria standard NTC). Per `T_R = 50` anni si ha `c_r = 1`. La formula e' valida nell'intervallo `5 <= T_R <= 500` anni; fuori intervallo la skill rifiuta il calcolo.

## par. 3.3.7 - Coefficiente di esposizione c_e(z)

```
c_e(z) = k_r^2 * c_t * ln(z/z_0) * [7 + c_t * ln(z/z_0)]   per z >= z_min
c_e(z) = c_e(z_min)                                          per z <  z_min   (eq. 3.3.5)
```

I parametri `k_r`, `z_0`, `z_min` sono dati dalla Tab. 3.3.II in funzione della **categoria di esposizione** del sito (I, II, III, IV, V):

| Categoria | k_r  | z_0 (m) | z_min (m) |
|-----------|------|---------|-----------|
| I         | 0.17 | 0.01    | 2.0       |
| II        | 0.19 | 0.05    | 4.0       |
| III       | 0.20 | 0.10    | 5.0       |
| IV        | 0.22 | 0.30    | 8.0       |
| V         | 0.23 | 0.70    | 12.0      |

Il coefficiente topografico `c_t` vale 1 per superfici a pendenza media inferiore al 3% (default skill). La categoria di esposizione si assegna mediante Tab. 3.3.III (rugosita' del territorio + zona vento + distanza dalla costa + altitudine): la skill non automatizza l'assegnazione, va dichiarata dall'utente.

## par. 3.3.8 - Coefficiente di forma c_p

Il coefficiente di forma `c_p` (o aerodinamico) e' funzione della tipologia e geometria della costruzione e dell'orientamento rispetto al vento. NTC fornisce valori indicativi per geometrie correnti; la Circolare 7/2019 par. C3.3 rinvia a CNR-DT 207/2008 per il dettaglio. **La skill non incorpora c_p**: lo richiede come input dell'utente, calcolato a parte dal progettista.

## par. 3.3.9 - Coefficiente dinamico c_d

`c_d` tiene conto degli effetti riduttivi associati alla non contemporaneita' delle massime pressioni e degli effetti amplificativi dovuti alle vibrazioni della struttura. NTC consente `c_d = 1` per:
- edifici di forma regolare con altezza inferiore a 80 m
- edifici industriali con copertura piana o a falde regolari

Per strutture sensibili (alti edifici, ciminiere, strutture leggere flessibili) si rinvia ad analisi dinamica (par. 3.3.9 e Circolare C3.3). **La skill usa `c_d = 1` come default** e non esegue analisi dinamica.

## Note sull'uso da parte della skill

- `v_b_0`, `a_0`, `k_s`: input utente (Tab. 3.3.I di NTC)
- `a_s`: input utente
- Categoria esposizione I/II/III/IV/V: input utente (dichiarata, non calcolata)
- `c_t`: default 1.0
- `c_p`: input utente
- `c_d`: default 1.0
- `T_R`: default 50 anni
- Output: `v_b`, `c_r`, `v_r`, `q_r`, `c_e`, `p` (in N/m^2 e kN/m^2)
