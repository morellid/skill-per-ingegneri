# Input - Fronte di scavo in trincea

Un progettista deve impostare due scavi secondo le NTC 2018 §6.8.6:

- **Scavo A**: una **trincea a fronte verticale** profonda **3,0 m**, nella quale e' prevista la **permanenza di
  personale** per la posa di una condotta.
- **Scavo B**: uno **scavo a sezione ristretta** di **1,5 m**, a fronte verticale, senza permanenza di personale
  e lontano da manufatti.

Chiede: in quali casi le NTC 2018 §6.8.6 impongono una **struttura di sostegno delle pareti di scavo**? Con
quali verifiche?
