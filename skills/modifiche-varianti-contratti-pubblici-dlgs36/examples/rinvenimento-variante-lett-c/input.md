# Input - rinvenimento imprevisto durante uno scavo

ha portato alla luce un **sottoservizio non mappato** non prevedibile con la dovuta diligenza in
fase di progettazione. Serve una variante che comporta un maggior costo di **250.000 euro**.
Rientra nei casi dell'art. 120 del D.Lgs. 36/2023? È entro i limiti? Chi la autorizza e cosa va
comunicato all'ANAC? Cita i commi.
