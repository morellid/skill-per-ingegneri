# Input - opzione di proroga prevista nel bando

Un appalto pubblico di servizi è in scadenza. Nel **bando e nei documenti di gara** era prevista
un'**opzione di proroga**. La stazione appaltante vuole avvalersene per proseguire il servizio.
È una modifica ammessa senza nuova gara? A quali prezzi e condizioni esegue il contraente? Cita
l'art. 120.
