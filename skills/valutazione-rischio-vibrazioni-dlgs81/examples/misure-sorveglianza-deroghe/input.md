# Input - superamento del valore limite e deroga

Per una mansione con macchina mobile fuoristrada, nonostante le misure adottate, il
tecnico rileva un'esposizione **corpo intero A(8) = 1,1 m/s²** (oltre il valore
limite). L'azienda opera nel **settore della navigazione marittima**.

Domande:
1. Cosa deve fare il datore di lavoro se il valore limite è superato (art. 203)?
2. È possibile una deroga e a quali condizioni (art. 205)?
3. Che sorveglianza sanitaria si applica (art. 204)?
