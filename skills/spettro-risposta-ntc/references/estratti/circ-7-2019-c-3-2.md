## par. C3.2 - Azione sismica (commento applicativo)

Il par. C3.2 della Circolare ribadisce e illustra con esempi numerici i passaggi di NTC 3.2:

1. **Determinazione di V_R = V_N * C_U**, con il clamp a 35 anni esplicitato.

2. **Determinazione dei T_R** per ciascuno stato limite tramite la relazione T_R = - V_R / ln(1 - P_VR), con P_VR da Tab. 3.2.I.

3. **Lookup dei parametri di pericolosita'** a_g, F_0, T_C* sul reticolo INGV (interpolazione spaziale), poi interpolazione logaritmica sui T_R.

4. **Determinazione di S_S e C_C** con le formule di Tab. 3.2.IV (clamp ai limiti di tabella) e di S_T (Tab. 3.2.V).

5. **Determinazione dei periodi caratteristici** T_B, T_C, T_D con le formule 3.2.7 e 3.2.8 di NTC.

6. **Costruzione dello spettro elastico** Se(T) ramo per ramo come da eq. 3.2.4.

## par. C3.2 (Appendice procedurale) - Note rilevanti per la skill

- La Circolare conferma l'interpolazione **logaritmica** sui T_R (sia per T_R che per p_k).
- Per **stati limite di esercizio** (SLO, SLD): la Circolare richiama che il valore di a_g, F_0, T_C* va comunque ricavato per il T_R di pertinenza, anche se SLO/SLD ricadono spesso nel ramo basso (T_R prossimo a 30-50 anni).
- Per **categoria topografica** non in sommita' del rilievo, la Circolare richiama l'interpolazione lineare di S_T tra base (S_T = 1.0) e sommita' (valore di Tab. 3.2.V), proporzionalmente all'altezza relativa del punto. La skill non automatizza questa correzione: lascia il warning operativo.

## Esempio numerico ricostruito (Circolare, sito di esempio)

> NB: le Circolari MIT contengono diversi esempi puntuali. Quello riportato sotto e' un *esempio sintetico* coerente con le formule (non riproduce un caso specifico della Circolare). Per la verifica vs CSLP fare riferimento a `tasks/run-test-suite.md`.

Sito generico, V_N = 50 anni, classe d'uso II -> V_R = 50 anni.
Categoria di sottosuolo C, categoria topografica T1, smorzamento xi = 5% -> eta = 1.

Parametri di pericolosita' al sito (esempio):

| T_R (anni) | a_g (g) | F_0    | T_C* (s) |
|------------|---------|--------|----------|
| 30         | 0.030   | 2.50   | 0.20     |
| 50         | 0.045   | 2.55   | 0.22     |
| 72         | 0.061   | 2.60   | 0.24     |
| 101        | 0.080   | 2.62   | 0.26     |
| 140        | 0.105   | 2.65   | 0.28     |
| 201        | 0.135   | 2.68   | 0.30     |
| 475        | 0.218   | 2.72   | 0.32     |
| 975        | 0.297   | 2.74   | 0.34     |
| 2475       | 0.420   | 2.76   | 0.36     |

Per SLV (T_R ~ 475 anni):
- a_g = 0.218 g, F_0 = 2.72, T_C* = 0.32 s
- S_S = 1.70 - 0.60 * 2.72 * 0.218 = 1.344 (in [1.00, 1.50] -> ok)
- C_C = 1.05 * 0.32^-0.33 = 1.529
- T_C = 1.529 * 0.32 = 0.489 s
- T_B = 0.489 / 3 = 0.163 s
- T_D = 4 * 0.218 + 1.6 = 2.472 s
- S = S_S * S_T = 1.344
- a_g * S * eta * F_0 = 0.218 * 1.344 * 1.0 * 2.72 = 0.797 g (plateau Se in T_B <= T <= T_C)

Questi valori coincidono (entro tolleranza di interpolazione) con l'output del modulo Python esposto in `examples/caso-conforme-fittizio-cu2-c-t1/` ed e' il principale dataset di confronto della test suite (`test_calcola_parametri_struttura`).
